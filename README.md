# 📊 Linear Programming Solver (Streamlit App)

An interactive Linear Programming Solver built using Python and Streamlit.  
This application allows users to define, solve, and visualize Linear Programming (LP) problems easily.

---

## 🚀 Features

- Supports **Maximization** and **Minimization**
- Dynamic number of variables (1–10)
- Dynamic number of constraints (1–10)
- Supports:
  - `<=` constraints
  - `>=` constraints
  - `=` constraints
- Option to allow negative variables
- Uses **SciPy (HiGHS)** solver
- Feasible region visualization (2-variable problems)
- Displays:
  - Optimal variable values
  - Optimal objective value
  - Feasible vertices and their objective values
- Clear All button to reset inputs

---

## 🛠️ Technologies Used

- Python
- Streamlit
- NumPy
- SciPy
- Matplotlib
